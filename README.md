# Flatland

A 2D puzzle-platformer game exploring dimensional shifting mechanics.

## Project Details
- **Platform**: itch.io + Android (Google Play Store)
- **Engine**: Godot 4.x
- **Target Launch**: January 25, 2026

## Concept
Players navigate a 2D world where they can shift between dimensions, revealing hidden platforms and altering the environment. Inspired by the novella "Flatland" by Edwin Abbott.

## Features (MVP)
- 10 playable levels
- Dimensional shift mechanic
- Physics-based platforming
- Environmental puzzles (doors, pressure plates)
- NPC interactions with dialogue
- Sound effects and music
- Main menu and level select
